## 图智乐BI数据分析平台

### 项目介绍

图智乐BI数据分析平台是基于 Spring Boot + MQ + AIGC 的智能数据分析平台。区别于传统 BI，用户只需要导入原始数据集、并输入分析诉求，就能自动生成可视化图表及分析结论。

### 功能点

1. 调用第三方AI，根据需求进行图表分析
2. 使用分表对用户Excel表中的大数据进行相应的存储
3. 采用三种方式生成图表
   - 同步
   - 线程池
   - 异步(MQ)
4. 使用redission中的RateLimiter令牌桶算法进行对用户的限流
